📋 Architektúra Rétegek

  • Entity Layer: User.java - Domain objektumok
  • Repository/DAO Layer: UserRepository, H2UserRepository - Adatbázis műveletek
  • Service Layer: UserService, UserServiceImpl - Üzleti logika
  • Controller Layer: UserResource, HelloResource - REST API végpontok
  • Presentation Layer: HelloBean - JSF felület
